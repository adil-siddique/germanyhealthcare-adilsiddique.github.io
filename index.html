Insight into Germany's Healthcare and Arthroplasty

The dataset for this project is real-world hospital quality reports from Germany. The data was mined from 9,000+ Quality reports from year 2020-2023. This project is focused on extracting real-world healthcare data and deriving business insights.

  • Automated file downloads: No bulk download option existed for the download. So, download was automated using Selenium based browser automation.
  • Automated Python data mining: A robust Python pipeline parses institutional identifiers, detects German OPS procedure codes, extracts surgery numbers for each OPS code, and aggregates all records into a clean, analysis-ready CSV.
  • Interactive Tableau Dashboards: Visualize the healthcare landscape with an overview dashboard and a specialized dashboard focusing on arthroplasty surgeries.
  • Data-Driven Insights: Enables data driven desicion making for stakeholders interested in the German healthcare and orthopedic market.
